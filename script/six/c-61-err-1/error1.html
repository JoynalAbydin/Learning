JavaScript error handle part 1

জাভাস্ক্রিপ্ট কোডে একটি ইরোর থাকলে পুরা কোড কাজ করা বন্ধ করে দেয়। যার কারনে ইরোর হ্যান্ডেল করতে হয়।

try-catch Statements:

try ব্লকের মধ্যে কোড রান করতে, যদি কোডে কোন ত্রুটি থাকে তবে সেটি ক্যাচ ব্লকে চলে যাবে, এবং ক্যাচে যা আছে সোটা দেখাবে

try {
  // যেকোনো ধরনের কোড রান করা যাবে। যার জন্য আপনি এরর Error হ্যান্ডল করতে চান।
} catch (error) {
  // এখানে এরর হ্যান্ডলিং কোড লিখুন
  console.error('An error occurred:', error);
}
