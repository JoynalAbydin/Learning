Focus Event Object

ফোকাস ইভেন্ট হলো একটি ওয়েব ইভেন্ট যা একটি HTML এলিমেন্ট (যেমন input, textarea, ইত্যাদি) ফোকাস পেতে অথবা থাকার সময় পাওয়া যায়। যখন ব্যবহারকারী কোনো ইনপুট ফিল্ডে ফোকাস করেন, তখন ফোকাস ইভেন্ট সক্রিয় হয়।

ফোকাস ইভেন্ট ব্যবহার করে আমরা ব্যবহারকারীর ইনপুটের ক্ষেত্রে কি ঘটছে সেটা জানতে পারি, যেমন ব্যবহারকারী কোনো ফিল্ডে প্রবেশ করলে সেটির রং পরিবর্তন করা, কোনো অবজেক্ট তৈরি করা ইত্যাদি।

Text Input Fild

এই কোডটি ব্যবহারকারীর ইনপুট করা টেক্সট এর সম্প্রদানের সাথে সাথে টেক্সটের সব অক্ষর আপারকেস (বড় হাতের) করে দেয়। 
 
    1. `const input = document.querySelector("input");`: এই লাইনে, আমরা DOM এলিমেন্ট কে ধরে নিয়ে আসছি। এখানে `querySelector` মেথডের মাধ্যমে আমরা প্রথম `` এলিমেন্টটি ধরে নিয়েছি।
       
    2. `input.addEventListener("input", function (e) { ... });`: এখানে, আমরা `` এলিমেন্টের "input" ইভেন্টের একটি ইভেন্ট লিসেনার যোগ করছি। "input" ইভেন্ট হলো ব্যবহারকারী ইনপুট ফিল্ডে কোনো পরিবর্তন করলে ঘটে। যখন ইনপুট ফিল্ডে কোনো টেক্সট ইনপুট করা হয়, এই ফাংশনটি কল হয়।
    
    3. `input.value = e.target.value.toUpperCase();`: ফাংশনের মধ্যে, আমরা সরাসরি `input` এলিমেন্টের `value` প্রপার্টিতে ব্যবহারকারীর ইনপুটের মানটি (টেক্সট) এক্সেস করেছি (`e.target.value`) এবং সেটির সব অক্ষর আপারকেসে (বড় হাতের) করে দেয়ার জন্য `toUpperCase()` মেথড ব্যবহার করেছি। সো ফলে, ব্যবহারকারী যেকোনো লেটার টাইপ করলে তা সরাসরি আপারকেসে রূপান্তরিত হবে এবং ইনপুট ফিল্ডে দেখানো হবে।
